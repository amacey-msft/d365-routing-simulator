# Changelog

## v1.0.0.1 — 2026-05-21

Initial packaged release of the D365 Contact Center Routing Simulator as a Power Platform unmanaged solution.

### What's Included
- `D365RoutingSimulator_1_0_0_1.zip` — Power Apps unmanaged solution, import-ready
- Interactive routing simulator web resource for testing D365CC routing scenarios
- Populate routing fields and validate routing rules against your live environment

### Install
1. Download `D365RoutingSimulator_1_0_0_1.zip` from the [Releases](../../releases) page
2. Import into your Power Platform environment via **Solutions → Import Solution**
3. Select **Unmanaged** when prompted
